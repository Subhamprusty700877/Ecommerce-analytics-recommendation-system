# E-Commerce Data Warehouse & Recommendation System

This project implements a small Data Warehouse and
product co-purchase recommendation analysis using
Snowflake, Python ETL and Power BI.

## Tools Used
- Python
- Snowflake
- SQL
- Power BI

## Project Features
- Star schema (Fact and Dimension tables)
- ETL using Python
- Sales analytics
- Product bought-together recommendation
- Power BI dashboards

## Folder Structure
- etl/       → Python ETL script
- sql/       → SQL scripts
- powerbi/   → Power BI report

## How to run
1. Load CSV data
2. Run Python ETL script
3. Execute SQL scripts in Snowflake
4. Open Power BI file
